# Axolop CRM V1.0 - Non-Functional Features (Locked)

## Overview
This document lists all features that will not be functional in the V1.0 release and will be locked with the temporary lock UI. These features will display a lock icon with "Coming in V1.1" tooltip.

## Non-Functional Features for V1.0 Release

### 1. Email Marketing Module
- **Location:** `/app/email-marketing`, `/app/email-marketing/campaigns/create`, `/app/email-marketing/workflows/create`, `/app/email-marketing/workflows/:workflowId`
- **Features not included:**
  - Email campaign creation and management
  - Email template builder
  - Email analytics and tracking
  - Automated email sequences
  - SendGrid integration

### 2. Inbox Section
- **Location:** `/app/inbox` (under Sales in sidebar)
- **Features not included:**
  - Email inbox functionality
  - Email integration
  - Email communication management

### 3. Live Call Features
- **Location:** `/app/live-calls`, `/app/calls` (under Sales > Conversations in sidebar)
- **Features not included:**
  - In-app calling functionality
  - Call recording and transcription
  - Call analytics dashboard
  - Call scheduling integration
  - AI-powered call scripts

### 4. Custom View Features
- **Location:** `/app/leads/custom-view` (under Sales > Leads in sidebar)
- **Features not included:**
  - Custom view creation for leads
  - Advanced filtering and views
  - Custom lead management views

### 5. Calendar Features (Advanced)
- **Location:** Calendar integration components
- **Features not included:**
  - Full calendar synchronization
  - Advanced scheduling features
  - Meeting automation
  - Calendar analytics

### 6. Second Brain AI Features
- **Location:** `/app/second-brain`, Second Brain button in main sidebar
- **Features not included:**
  - AI-powered knowledge management
  - Local AI processing
  - Advanced mind mapping
  - AI-assisted content creation

### 6. Service Category Features
- **Location:** Service section in sidebar (`/app/tickets`, `/app/knowledge-base`, `/app/customer-portal`, `/app/support-analytics`)
- **Features not included:**
  - Ticket management system
  - Knowledge base functionality
  - Customer portal
  - Support analytics

### 7. Advanced Workflow Features
- **Location:** Advanced workflow components
- **Features not included:**
  - Complex multi-step workflows
  - Advanced AI triggers
  - Enterprise integrations

### 8. Advanced Reporting
- **Location:** `/app/reports/activity-overview`, `/app/reports/activity-comparison`, `/app/reports/opportunity-funnels`, `/app/reports/status-changes`, `/app/reports/explorer` (under Sales > Activities in sidebar)
- **Features not included:**
  - Custom report builder
  - Advanced analytics
  - Predictive analytics

## Lock Implementation Details

### Lock UI Components
All non-functional features will display:
- Lock overlay on the component
- "Coming in V1.1" tooltip on hover
- `cursor-not-allowed` styling
- Grayed out appearance to indicate inaccessibility

### Implementation Style
The lock implementation will match the existing temporary lock style used on:
- Chat button in main sidebar
- Tasks button in main sidebar
- Second Brain button in main sidebar

## Future V1.1 Features
These features will be fully implemented and unlocked in the V1.1 release:
- Complete email marketing suite
- Live call functionality
- Advanced calendar features
- Second Brain AI features
- Full service category functionality

## Quality Assurance
- All locked features should display consistently with the temporary lock design
- Tooltips should appear on hover with clear messaging
- Navigation to locked features should be prevented
- UI should clearly communicate that features are coming in V1.1
