# AxolopCRM Brand Guidelines

## Brand Identity

AxolopCRM is a sophisticated, professional CRM platform designed for modern sales teams. Our brand identity reflects this with a clean, powerful aesthetic that combines classic black with vibrant red accents.

## Color Palette

### Primary Colors
- **Main Black:** `#101010` 
  - Used for all primary text, backgrounds, and core interface elements
  - Provides a professional, high-contrast foundation for the UI
- **Accent Color:** `#7b1c14` 
  - Used for highlights, calls to action, and key interactive elements
  - Creates visual interest and guides user attention to important areas

### Secondary Colors
These colors are used for category-specific areas within the CRM:

- **Sales Category:** `#4C7FFF` (primary blue)
- **Marketing Category:** `#00D084` (primary green) 
- **Service Category:** `#FFB800` (primary yellow)

### Neutral Colors
- **Light Gray:** `#F5F5F5` (backgrounds, subtle elements)
- **Medium Gray:** `#999999` (secondary text, borders)
- **Dark Gray:** `#666666` (tertiary text)

## Logo Usage

### Logo Variants
- **Primary Logo:** Full-color logo with transparent background
- **Inverse Logo:** White logo for dark backgrounds
- **Transparent Logo:** As provided in `/branding/LOGO/transparent logo.png`

### Logo Placement
- Maintain minimum clear space of 2x the logo height around all sides
- Do not stretch, skew, or alter the proportions of the logo
- Ensure the logo is always placed on a contrasting background for visibility

## Typography

### Primary Font
- **Font Family:** Inter (system font stack)
- **Usage:** All body text, headings, and interface elements
- **Weights:** 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

## Background Assets

### Primary Background
- Located at: `/branding/banner/background.png`
- Features the signature black and red gradient that represents our brand
- Should be used consistently across all branded entry points and marketing materials

## Application to UI Elements

### Buttons and Interactive Elements
- Primary buttons: `#7b1c14` (red accent) with white text
- Secondary buttons: `#101010` (black) with white text
- Hover states: Darken colors by 15% for visual feedback

### Forms and Inputs
- Input borders: `#999999` (medium gray) by default
- Focused state: `#4C7FFF` (blue) for primary action forms
- Error state: `#FF0000` (red) for form validation

### Navigation Elements
- Sidebar background: `#101010` (main black)
- Active navigation items: `#7b1c14` (red accent)
- Hover states: `#333333` (dark gray) for menu items

## Consistency Guidelines

All marketing materials, documentation, and user interfaces should consistently use the specified brand colors to maintain a cohesive experience across all touchpoints. The red accent should be used sparingly but effectively to draw attention to important actions and elements.

## Brand Voice and Messaging

Our brand voice is:
- Professional yet approachable
- Confident and knowledgeable
- Focused on empowering sales teams
- Emphasizing reliability and performance
