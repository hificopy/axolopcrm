## üéØ Testimonial Guidelines

### What Makes a Great Testimonial

1. **Specific Results**
   - Time saved
   - Money saved
   - Efficiency gains
   - ROI improvements

2. **Emotional Impact**
   - How it made their work easier
   - Stress reduction
   - Confidence boost
   - Peace of mind

3. **Comparative Value**
   - What they used before
   - Why they switched
   - What makes Axolop different
   - Why they stayed

4. **Use Case Clarity**
   - Industry/business type
   - Team size
   - Specific features used
   - Main problem solved

### Testimonial Collection Process

1. **Identify Happy Customers**
   - High engagement
   - Long-term users
   - Business tier customers
   - Feature power users

2. **Request Testimonial**
   - Email template
   - Specific questions
   - Media requests
   - Incentive offer (if applicable)

3. **Review & Edit**
   - Get permission
   - Format for clarity
   - Extract best quotes
   - Create multiple versions

4. **Publish & Promote**
   - Add to website
   - Share on social media
   - Include in email campaigns
   - Feature in sales materials
