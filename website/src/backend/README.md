# Backend Services

This directory contains all server-side code for the CRM application.

## Structure

- `controllers/` - Request handlers and business logic
- `middleware/` - Authentication and validation layers
- `routes/` - API endpoint definitions
- `services/` - Core business services
- `utils/` - Helper functions and utilities

## Backend Philosophy

Server-side development follows these principles:
- RESTful API design
- OAuth security with Auth0
- Prisma ORM integration
- Error-first callback handling
- Comprehensive logging
- Input validation
