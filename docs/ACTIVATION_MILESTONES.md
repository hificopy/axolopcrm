# ACTIVATION MILESTONES
## User Journey to Power User Status

---

# THE HORMOZI ACTIVATION PRINCIPLE

**Core Insight:** Users who STAY do specific things in their first 7-30 days that churned users DON'T.

**Our Job:** Find those behaviors and build features that drive users to complete them.

---

# ACTIVATION RESEARCH FRAMEWORK

## Step 1: Segment Users

| Segment | Definition | Target % | Analysis Focus |
|---------|------------|----------|----------------|
| **Champions** | 6+ months, high usage, NPS 9-10 | 10% | What they did in Day 1-30 |
| **Survivors** | 3-6 months, moderate usage | 30% | What kept them vs churned |
| **Churned** | <3 months, canceled | 60% | What they didn't do |

# DETAILED MILESTONE DEFINITIONS

## M1: First Win (Day 1)

**Target Action:** Create first contact in Axolop

**Why It Matters:**
- Proves commitment beyond "just browsing"
- First data entered = emotional investment
- Without contacts, CRM is useless

**Success Criteria:**
- [ ] Contact has name
- [ ] Contact has email or phone
- [ ] Created within 60 minutes of signup

**Current State:**
- Estimated completion: ~65%
- Drop-off: ~35%

**Features to Drive M1:**
| Feature | Priority | Impact |
|---------|----------|--------|
| Pre-filled sample contacts | NOW | +15% |
| "Add your first contact" prompt | NOW | +10% |
| CSV import wizard | NOW | +20% |
| Google Contacts import | NEXT | +25% |

## Cohort Analysis Required

```sql
-- Query to identify activation patterns
SELECT
  user_cohort_week,
  milestone,
  pct_completed,
  median_time_to_complete,
  churn_rate_if_not_completed,
  retention_90_day
FROM activation_analysis
GROUP BY user_cohort_week, milestone
ORDER BY user_cohort_week DESC, milestone ASC;
```
