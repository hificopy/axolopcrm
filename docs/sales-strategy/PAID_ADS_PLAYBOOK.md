## AD CREATIVE FRAMEWORK

### The PASO Formula

**P** - Problem (Hook them with pain)
**A** - Agitate (Make it worse)
**S** - Solution (Present Axolop)
**O** - Offer (CTA with urgency)

### The 3-Second Rule

You have 3 seconds to stop the scroll. Front-load the most painful/interesting part.

## AD TESTING FRAMEWORK

### Phase 1: Hook Testing (Week 1)

Test 3 different hooks with same offer:

| Ad | Hook |
|----|------|
| A | "I lost $47k because I forgot to follow up" |
| B | "Stop paying $500/mo for CRMs that don't work" |
| C | "Your competitor is closing deals you're losing" |

**Budget:** $33/day each = $100/day total
**Metric:** CTR (Click-Through Rate)
**Winner:** Highest CTR

---

### Phase 2: Offer Testing (Week 2)

Use winning hook, test 3 offers:

| Ad | Offer |
|----|-------|
| A | "14-day free trial, no credit card" |
| B | "30% more deals in 90 days or it's free" |
| C | "$99/month - save $1,000+ vs current tools" |

**Budget:** $33/day each = $100/day total
**Metric:** Conversion Rate (Signup)
**Winner:** Highest conversion rate

---

### Phase 3: Creative Testing (Week 3)

Use winning hook + offer, test 3 creatives:

| Ad | Creative |
|----|----------|
| A | Talking head video (founder) |
| B | Screen recording demo |
| C | Before/after carousel |

**Budget:** $33/day each = $100/day total
**Metric:** Cost Per Acquisition (CPA)
**Winner:** Lowest CPA

---

### Phase 4: Audience Testing (Week 4)

Use winning hook + offer + creative, test 3 audiences:

| Ad | Audience |
|----|----------|
| A | Agency owners, 25-55, marketing interests |
| B | Sales professionals, job titles |
| C | Lookalike of existing customers |

**Budget:** $33/day each = $100/day total
**Metric:** Return on Ad Spend (ROAS)
**Winner:** Highest ROAS

---

### Scaling Phase (Week 5+)

Take winning combination:
- Winning hook
- Winning offer
- Winning creative
- Winning audience

Scale budget 20% per day until performance drops.

**Example:**
- Day 1: $100/day (winner established)
- Day 2: $120/day
- Day 3: $144/day
- Day 4: $173/day
- Day 5: $207/day

Watch CPAs. If CPA increases 20%+, plateau at previous budget.

## TRACKING & METRICS

### Key Performance Indicators

| Metric | Target | How to Improve |
|--------|--------|----------------|
| CTR (Click-Through Rate) | 1.5%+ | Better hooks, better targeting |
| CPC (Cost Per Click) | <$3 | Better quality score, relevance |
| CPL (Cost Per Lead) | <$50 | Better landing page, offer |
| Trial Signup Rate | 30%+ | Landing page optimization |
| Trial-to-Paid Rate | 30%+ | Onboarding, sales process |
| CAC (Customer Acquisition Cost) | <$200 | Full-funnel optimization |
| ROAS (Return on Ad Spend) | 3:1+ | Everything |
